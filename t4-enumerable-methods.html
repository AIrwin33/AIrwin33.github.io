How to use the enumerable method group_by
Technical Blog
Andrew Irwin
November 22, 2014

The group_by method takes an object and block and returns a hash that contains groups organized by whatever designation you assigned(size, length, alphabetical). This is a simple way of taking an array and passing it to a hash. The elements of the array are then passed to the values of the hash, while whatever organizational assignment was given passed to a key.

The code should look something like this:

object.group_by {|object| block}

Lets say we wanted to sort an array of types of fruit by the length of the string.

Example:

fruit = ["apples", "tomatoes", "grapefruit", "bananas", "papayas", "mangoes", "blackberries", "grapes"]
Note on sorting: don't forget that the computer organizes uppercase and lowercase words differently. make sure your strings are lowercase or use the downcase method before sorting.
p fruit.group_by {|fruit| fruit.length}
The result groups each fruit by its string length.
{6=>["apples", "grapes"], 8=>["tomatoes"], 10=>["grapefruit"], 7=>["bananas", "papayas", "mangoes"], 12=>["blackberries"]}
